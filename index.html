  1. A. йӨҳејҰе®ҡеҫӢ (Law of Cosines)

жң¬зүҲ v0.1 е…Ҳе®ҢжҲҗдё»йЎҢ 1вҖ“5пјҢдё»йЎҢз·Ёиҷҹз¶ӯжҢҒз©©е®ҡгҖӮ

1. з·ҡжҖ§д»Јж•ё вҖ” еҗ‘йҮҸгҖҒзҹ©йҷЈгҖҒе…§з©Қ

зҘһз¶“з¶Іи·Ҝзҡ„жҜҸдёҖеұӨпјҢжң¬иіӘе°ұжҳҜзҹ©йҷЈд№ҳжі• + йқһз·ҡжҖ§иҪүжҸӣгҖӮе…ҲжҠҠиӘһиЁҖеӮҷйҪҠгҖӮ

1.1 еҗ‘йҮҸиҲҮе…§з©Қ вҖ” жҠ•еҪұиҲҮйҮҸжё¬е°ҚйҪҠзЁӢеәҰ

дёҖеҖӢ $n$ з¶ӯеҗ‘йҮҸ $\mathbf{x}=(x_1,x_2,\dots,x_n)^\top$ еҸҜзңӢжҲҗдёҖеҖӢиіҮж–ҷй»һзҡ„зү№еҫөпјҲдҫӢеҰӮдёҖжўқ flow зҡ„е°ҒеҢ…ж•ёгҖҒbytesгҖҒжҷӮй•·вҖҰпјүгҖӮе…§з©Қ (dot product) жҳҜжңҖж ёеҝғзҡ„йҒӢз®—пјҡ

$$\mathbf{w}\cdot\mathbf{x} = \mathbf{w}^\top\mathbf{x} = \sum_{i=1}^{n} w_i x_i$$

е®ғиЎЎйҮҸе…©еҗ‘йҮҸзҡ„гҖҢе°ҚйҪҠзЁӢеәҰгҖҚгҖӮе№ҫдҪ•дёҠ $\mathbf{w}^\top\mathbf{x} = \lVert\mathbf{w}\rVert\,\lVert\mathbf{x}\rVert\cos\phi$пјҢе…¶дёӯ $\lVert\mathbf{x}\rVert=\sqrt{\sum_i x_i^2}$ жҳҜй•·еәҰгҖҒ$\phi$ жҳҜеӨҫи§’гҖӮдёҖеҖӢзҘһз¶“е…ғеҒҡзҡ„е°ұжҳҜдёҖж¬Ўе…§з©Қ $\mathbf{w}^\top\mathbf{x}$ еҠ еҒҸзҪ® $b$пјҢд»ЈиЎЁгҖҢз”Ёж¬ҠйҮҚйҮҸжё¬ијёе…ҘжңүеӨҡз¬ҰеҗҲжҹҗжЁЈејҸгҖҚгҖӮпјҲжіЁпјҡйҖҷиЈЎзҡ„гҖҢе°ҚйҪҠгҖҚжҳҜж–№еҗ‘иҲҮй•·еәҰзҡ„з¶ңеҗҲеҲҶж•ёпјӣзҙ”ж–№еҗ‘зҡ„зӣёдјјеәҰйңҖе…ҲжӯЈиҰҸеҢ–пјҢиҰӢ 1.5гҖӮпјү

е№ҫдҪ•ж„Ҹзҫ©гҖӮ е…§з©Қ = гҖҢ$\mathbf{a}$ жҠ•еҪұеҲ° $\mathbf{b}$ ж–№еҗ‘зҡ„й•·еәҰ $\lVert\mathbf{a}\rVert\cos\phi$гҖҚеҶҚд№ҳдёҠ $\lVert\mathbf{b}\rVert$гҖӮеӨҫи§’и¶Ҡе°ҸгҖҒжҠ•еҪұи¶Ҡй•·гҖҒе…§з©Қи¶ҠеӨ§гҖӮ

еҖјжҖҺйәји§Ји®ҖгҖӮ е…§з©ҚдёҚжҳҜгҖҢеӨ§ / е°ҸгҖҚе…©жҘөпјҢиҖҢжҳҜдёҖжўқжңүжӯЈиІ зҡ„и»ё вҖ” и¶ҠжӯЈи¶Ҡе°ҚйҪҠгҖҒиҝ‘ 0 дёҚзӣёй—ңгҖҒи¶ҠиІ и¶ҠзӣёеҸҚпјҡ

е…§з©ҚеҖјеӨҫи§’ ПҶcos ПҶе°ҚйҪҠзӢҖж…Ӣ
жңҖеӨ§пјҲжӯЈпјү0В°+1е®Ңе…ЁеҗҢеҗ‘пјҲжңҖе°ҚйҪҠпјү
вүҲ 090В°0еһӮзӣҙгҖҒдёҚзӣёй—ң
жңҖе°ҸпјҲиІ пјү180В°вҲ’1е®Ңе…ЁеҸҚеҗ‘пјҲжңҖдёҚе°ҚйҪҠпјү

жіЁж„Ҹпјҡе…§з©Қ $=\lVert\mathbf{a}\rVert\lVert\mathbf{b}\rVert\cos\phi$пјҢеҖјеӨ§д№ҹеҸҜиғҪеҸӘжҳҜеҗ‘йҮҸй•·пјӣиҰҒзҙ”жҜ”ж–№еҗ‘е°ҚйҪҠгҖҒдёҚеҸ—й•·еәҰе№Іж“ҫпјҢзңӢ $\cos\phi\in[-1,1]$пјҲжӯЈиҰҸеҢ–еҫҢзҡ„е…§з©ҚпјҢиҰӢ 1.5пјүгҖӮзҘһз¶“е…ғеҗҢзҗҶпјҡијёе…Ҙи¶Ҡе°ҚйҪҠж¬ҠйҮҚжЁЈејҸпјҢе…§з©Қи¶ҠеӨ§гҖҒжҝҖжҙ»и¶Ҡеј·пјҲиІ еҖјпјқеј·зғҲдёҚз¬ҰеҗҲпјүгҖӮ

е…§з©Қзҡ„д»Јж•ёе®ҡзҫ© $\sum_i a_ib_i$ иҲҮе№ҫдҪ•е®ҡзҫ© $\lVert\mathbf{a}\rVert\lVert\mathbf{b}\rVert\cos\phi$ зӮәдҪ•жңғзӣёзӯүпјҹй—ңйҚөжҳҜйӨҳејҰе®ҡеҫӢпјӣе®Ңж•ҙжҺЁе°ҺиҰӢйҷ„йҢ„ AгҖӮ

1.2 зҘһз¶“е…ғ вҖ” жҠҠз·ҡжҖ§д»Јж•ёи®ҠжҲҗгҖҢжңғеӯёзҝ’зҡ„зү№еҫөеҒөжё¬еҷЁгҖҚ

дәәе·ҘзҘһз¶“е…ғжҳҜж·ұеәҰеӯёзҝ’зҡ„жңҖе°Ҹе–®е…ғпјҢйқҲж„ҹдҫҶиҮӘз”ҹзү©зҘһз¶“е…ғпјҲж”¶еӨҡеҖӢијёе…ҘгҖҒеҠ ж¬Ҡж•ҙеҗҲгҖҒи¶…йҒҺй–ҖжӘ»жүҚгҖҢжҝҖзҷјгҖҚж”ҫйӣ»пјүгҖӮе®ғзҡ„ж•ёеӯёе®ҡзҫ©еҸӘжңүе…©жӯҘпјҡ

$$z = \mathbf{w}^\top\mathbf{x} + b = \sum_i w_i x_i + b, \qquad a = \sigma(z)$$

  • $\mathbf{x}$пјҡијёе…ҘпјҲдёҠдёҖеұӨијёеҮәжҲ–еҺҹе§Ӣзү№еҫөпјү
  • $\mathbf{w}$пјҡж¬ҠйҮҚпјҲеҸҜеӯёпјӣд»ЈиЎЁгҖҢйҖҷзҘһз¶“е…ғеңЁжүҫд»ҖйәјжЁЈејҸгҖҚпјү
  • $b$пјҡеҒҸзҪ®пјҲеҸҜеӯёпјӣиӘҝж•ҙжҝҖзҷјй–ҖжӘ»пјү
  • $\sigma$пјҡжҝҖжҙ»еҮҪж•ёпјҲйқһз·ҡжҖ§пјҢеҰӮ ReLU / sigmoidпјҢиҰӢдё»йЎҢ 9пјү
  • $a$пјҡијёеҮәпјҲжҝҖжҙ»еҖјпјү

ж¬ҠйҮҚ $\mathbf{w}$ вҖ” зү№еҫөжЁЈжқҝгҖӮ $\mathbf{w}^\top\mathbf{x}$ жҳҜе…§з©ҚпјҲ1.1пјүпјҢйҮҸжё¬ијёе…ҘиҲҮгҖҢжЁЈејҸ $\mathbf{w}$гҖҚзҡ„е°ҚйҪҠзЁӢеәҰгҖӮжүҖд»Ҙж¬ҠйҮҚе°ұжҳҜйҖҷеҖӢзҘһз¶“е…ғеӯёеҲ°зҡ„зү№еҫөжЁЈжқҝпјҡијёе…Ҙи¶ҠеғҸ $\mathbf{w}$гҖҒе…§з©Қи¶ҠеӨ§гҖҒи¶ҠеҸҜиғҪжҝҖзҷј вҖ” йҖҷе°ұжҳҜгҖҢзҘһз¶“е…ғпјқзү№еҫөеҒөжё¬еҷЁгҖҚзҡ„з”ұдҫҶгҖӮ

ж¬ҠйҮҚеҫһе“ӘдҫҶпјҹ дёҠйқўзҡ„ $\mathbf{w}, b$ дёҚжҳҜй җиЁӯзҡ„пјҢиҖҢжҳҜиЁ“з·ҙеӯёеҮәдҫҶзҡ„пјҡеҲқе§ӢйҡЁж©ҹпјҢеҶҚз”ЁжўҜеәҰдёӢйҷҚ + еҸҚеҗ‘еӮіж’ӯдҫқжҗҚеӨұеҸҚиҰҶеҫ®иӘҝпјҲиҰӢдё»йЎҢ 3 / 6 / 10пјүгҖӮе№ҫдҪ•дёҠпјҢиЁ“з·ҙе°ұжҳҜдёҖзӣҙжҢӘеӢ•жұәзӯ–йӮҠз•ҢпјҲиӘҝж•ҙжі•еҗ‘йҮҸ $\mathbf{w}$ иҲҮеҒҸзҪ® $b$пјүпјҢеҫһйҡЁж©ҹеҲқе§ӢйҖҗжӯҘжҠҠдёҚеҗҢйЎһеҲҘеҲҶеҲ°жӯЈзўәзҡ„е…©еҒҙпјӣеӣ зӮәе…§з©ҚеңЁе°ҚйҪҠжҷӮжңҖеӨ§пјҢ$\mathbf{w}$ жңғиҮӘз„¶й•·жҲҗгҖҢйҖҷзҘһз¶“е…ғе°Ҳй–ҖеҒөжё¬зҡ„жЁЈејҸгҖҚвҖ”вҖ”йҖҷе°ұжҳҜгҖҢзү№еҫөжЁЈжқҝгҖҚзҡ„з”ұдҫҶпјҲиіҮж–ҷ + жҗҚеӨұй©…еӢ•зҡ„иЎЁеҫөеӯёзҝ’пјүгҖӮ

еҒҸзҪ® $b$ вҖ” й–ҖжӘ»гҖӮ е№ҫдҪ•дёҠ $\mathbf{w}^\top\mathbf{x}+b=0$ жҳҜијёе…Ҙз©әй–“иЈЎзҡ„дёҖеҖӢжұәзӯ–и¶…е№ійқўпјҲ2D жҳҜдёҖжўқз·ҡгҖҒ3D жҳҜе№ійқўпјүпјҢ$\mathbf{w}$ жҳҜе®ғзҡ„жі•еҗ‘йҮҸгҖӮ$b$ жұәе®ҡйҖҷе№ійқўйӣўеҺҹй»һеӨҡйҒ гҖҒеҚігҖҢеӨҡе®№жҳ“иў«жҝҖзҷјгҖҚгҖӮ$z=\mathbf{w}^\top\mathbf{x}+b$ еҸҜзңӢжҲҗијёе…Ҙй»һеҲ°и¶…е№ійқўзҡ„её¶з¬Ұиҷҹи·қйӣўпјҲд№ҳдёҠ $\lVert\mathbf{w}\rVert$пјүпјҡдёҖеҒҙзӮәжӯЈпјҲжҝҖзҷјпјүгҖҒеҸҰдёҖеҒҙзӮәиІ пјҲдёҚжҝҖзҷјпјүгҖӮ

жҝҖжҙ» $\sigma$ вҖ” йқһз·ҡжҖ§иҪүжҸӣгҖӮ иӢҘжІ’жңү $\sigma$пјҢеӨҡеұӨзҘһз¶“е…ғз–Ҡиө·дҫҶд»ҚжҳҜз·ҡжҖ§пјҲзҹ©йҷЈйҖЈд№ҳ $\mathbf{W}_2\mathbf{W}_1\mathbf{x}$ йӮ„жҳҜдёҖеҖӢзҹ©йҷЈпјүпјҢз–Ҡе№ҫеұӨйғҪеҸӘиғҪеӯёз·ҡжҖ§йӮҠз•ҢгҖӮеҠ е…Ҙйқһз·ҡжҖ§иҪүжҸӣеҫҢпјҢеӨҡеұӨжүҚиғҪйҖјиҝ‘д»»ж„ҸиӨҮйӣңеҮҪж•ё вҖ” йҖҷжҳҜж·ұеәҰеӯёзҝ’иғҪеӯёиӨҮйӣңжЁЈејҸзҡ„й—ңйҚөпјҲжҝҖжҙ»еҮҪж•ёзҙ°зҜҖиҰӢдё»йЎҢ 9пјүгҖӮ

жҺҘеӣһдёҠдёӢж–ҮгҖӮ дёҖеҖӢзҘһз¶“е…ғ = дёҖж¬Ўе…§з©Қ + еҒҸзҪ® + йқһз·ҡжҖ§иҪүжҸӣпјӣдёҖеұӨ = еҫҲеӨҡзҘһз¶“е…ғ = дёҖеҖӢзҹ©йҷЈ $\mathbf{W}$пјҲжҜҸеҲ—жҳҜдёҖеҖӢзҘһз¶“е…ғзҡ„ж¬ҠйҮҚпјҢеҚі 1.3 зҡ„еҲ—иҰ–и§’пјүпјҡ$\mathbf{z}=\mathbf{W}\mathbf{x}+\mathbf{b}$гҖҒ$\mathbf{a}=\sigma(\mathbf{z})$пјӣеӨҡеұӨе Ҷз–Ҡ + йқһз·ҡжҖ§иҪүжҸӣ = ж·ұеәҰз¶Іи·ҜпјҢеүҚдёҖеұӨзҡ„ијёеҮәз•¶дёӢдёҖеұӨијёе…ҘпјҢйҖҗеұӨжҠҪеҸ–и¶ҠдҫҶи¶ҠжҠҪиұЎзҡ„зү№еҫөгҖӮ

1.3 зҹ©йҷЈд№ҳжі• вҖ” еә§жЁҷи®ҠжҸӣпјҢд№ҹжҳҜе°Қеҗ„жЁЈжқҝзҡ„е°ҚйҪҠзЁӢеәҰ

жҠҠдёҖеұӨ $m$ еҖӢзҘһз¶“е…ғзҡ„ж¬ҠйҮҚз–ҠжҲҗзҹ©йҷЈ $\mathbf{W}\in\mathbb{R}^{m\times n}$пјҢдёҖж¬Ўз®—еҮәж•ҙеұӨеҠ ж¬Ҡе’Ңпјҡ

$$\mathbf{z} = \mathbf{W}\mathbf{x} + \mathbf{b},\qquad z_j = \sum_{i=1}^{n} W_{ji}x_i + b_j$$

$\mathbf{z}\in\mathbb{R}^m$ жҳҜйҖҷеұӨзҡ„еҺҹе§ӢијёеҮә (pre-activation)гҖӮдёҖж¬ЎеүҚеҗ‘еӮіж’ӯе°ұжҳҜйҖЈдёІзҡ„ $\mathbf{W}\mathbf{x}+\mathbf{b}$гҖӮйҖҷд№ҹжҳҜзӮәд»Җйәј GPU/NPU жңүз”Ё вҖ” е®ғеҖ‘зӮәеӨ§иҰҸжЁЎзҹ©йҷЈд№ҳжі• (GEMM) иҖҢз”ҹгҖӮ

з¶ӯеәҰиҰҸеүҮпјҲжҖҺйәјеҲӨж–·зөҗжһңеҪўзӢҖпјүгҖӮ зҹ©йҷЈ Г— еҗ‘йҮҸеҫ—еҲ°зҡ„жҳҜеҗ‘йҮҸпјҲдёҚжҳҜзҹ©йҷЈпјүпјҡ

$$\mathbf{W}_{m\times n}\,\mathbf{x}_{n\times 1} = \mathbf{z}_{m\times 1}$$

е…§еҒҙз¶ӯеәҰ $n$ еҝ…й ҲзӣёзӯүпјҲW ж¬„ж•ё = x з¶ӯеәҰпјүпјӣзөҗжһңжҳҜ $m\times 1$ ж¬„еҗ‘йҮҸпјҢй•·еәҰпјқW зҡ„еҲ—ж•ё $m$пјҲпјқи©ІеұӨзҘһз¶“е…ғж•ёпјүгҖӮеӣ зӮәжҜҸеҖӢ $z_j$ жҳҜгҖҢW з¬¬ $j$ еҲ— $\cdot$ xгҖҚзҡ„е…§з©ҚпјҲзҙ”йҮҸпјүпјҢ$m$ еҲ— вҶ’ $m$ еҖӢзҙ”йҮҸ вҶ’ $m$ з¶ӯеҗ‘йҮҸгҖӮ

йҒӢз®—з¶ӯеәҰзөҗжһң
зҹ©йҷЈ Г— еҗ‘йҮҸ$(m\times n)(n\times 1)$еҗ‘йҮҸ $m\times 1$
зҹ©йҷЈ Г— зҹ©йҷЈ$(m\times n)(n\times p)$зҹ©йҷЈ $m\times p$
еҗ‘йҮҸбөҖ Г— еҗ‘йҮҸпјҲе…§з©Қпјү$(1\times n)(n\times 1)$зҙ”йҮҸ
еҗ‘йҮҸ Г— еҗ‘йҮҸбөҖпјҲеӨ–з©Қпјү$(n\times 1)(1\times n)$зҹ©йҷЈ $n\times n$

пјҲеҗ‘йҮҸжң¬иә«еҚі $m\times 1$ зҹ©йҷЈпјӣзҹ©йҷЈ Г— еҗ‘йҮҸжҳҜзҹ©йҷЈд№ҳжі•зҡ„зү№дҫӢпјҢзөҗжһңеүӣеҘҪгҖҢдёҖж¬„гҖҚж•…зЁұеҗ‘йҮҸгҖӮпјү

е№ҫдҪ•ж„Ҹзҫ©гҖӮ зҹ©йҷЈд№ҳдёҖеҖӢеҗ‘йҮҸ = е°Қж•ҙеҖӢз©әй–“еҒҡдёҖж¬Ўз·ҡжҖ§и®ҠжҸӣпјҲж—ӢиҪү / зё®ж”ҫ / еүӘеҲҮ / зҝ»иҪүпјүгҖӮжңҖзӣҙиҰәзҡ„зңӢжі•пјҡзҹ©йҷЈзҡ„жҜҸдёҖгҖҢж¬„гҖҚе°ұжҳҜеҺҹжң¬зҡ„еә§жЁҷи»ёпјҲеҹәеә•пјүиў«и®ҠжҸӣеҫҢиҗҪеҲ°зҡ„дҪҚзҪ®гҖӮеӣ зӮә

$$\mathbf{W}\mathbf{x} = x_1\,\mathbf{w}_1 + x_2\,\mathbf{w}_2 + \dots + x_n\,\mathbf{w}_n$$

е…¶дёӯ $\mathbf{w}_j$ зӮә $\mathbf{W}$ зҡ„з¬¬ $j$ ж¬„гҖӮ$\mathbf{W}\mathbf{x}$ е°ұжҳҜгҖҢз”Ё $\mathbf{x}$ зҡ„еҲҶйҮҸпјҢжҠҠ $\mathbf{W}$ зҡ„еҗ„ж¬„пјҲи®ҠжҸӣеҫҢзҡ„и»ёпјүеҠ ж¬ҠзӣёеҠ гҖҚгҖӮз”ұеҹәеә•ејөжҲҗзҡ„е–®дҪҚжӯЈж–№еҪўпјҢжңғиў« $\mathbf{W}$ жӢүжҲҗдёҖеҖӢе№іиЎҢеӣӣйӮҠеҪўпјҢе…¶йӮҠе°ұжҳҜ $\mathbf{W}$ зҡ„ж¬„еҗ‘йҮҸпјҡ

е…©зЁ®дә’иЈңиҰ–и§’пјҡ

  • ж¬„иҰ–и§’пјҲз©әй–“и®ҠжҸӣпјүпјҡж¬„пјқеҹәеә•иҗҪй»һпјӣ$\mathbf{W}\mathbf{x}$пјқ$\mathbf{x}$ иў«и®ҠжҸӣеҫҢзҡ„ж–°дҪҚзҪ®гҖӮ
  • еҲ—иҰ–и§’пјҲзү№еҫөеҒөжё¬пјүпјҡжҜҸдёҖеҲ—жҳҜдёҖеҖӢж¬ҠйҮҚеҗ‘йҮҸпјҢ$z_j=\mathbf{w}_{j,:}^\top\mathbf{x}$ жҳҜе…§з©Қпјқ$\mathbf{x}$ е°Қз¬¬ $j$ еҖӢжЁЈејҸзҡ„е°ҚйҪҠеҲҶж•ё вҶ’ дёҖеұӨпјқдёҖжҺ’зү№еҫөеҒөжё¬еҷЁпјҲжҺҘеӣһ 1.2 зҘһз¶“е…ғпјүгҖӮ

зҹ©йҷЈзӣёд№ҳпјқи®ҠжҸӣзҡ„еҗҲжҲҗпјҡ$\mathbf{A}(\mathbf{B}\mathbf{x})$ жҳҜгҖҢе…ҲеҒҡ B еҶҚеҒҡ AгҖҚгҖӮзү№ж®Ҡжғ…еҪўпјҡе°Қи§’зҹ©йҷЈпјқжІҝеҗ„и»ёзё®ж”ҫпјҲжҺҘеӣһ 1.6 зү№еҫөеҖјпјүгҖҒж—ӢиҪүзҹ©йҷЈпјқиҪүдёҖеҖӢи§’еәҰгҖҒиЎҢеҲ—ејҸпјқйқўз©Қ / й«”з©Қиў«ж”ҫеӨ§зҡ„еҖҚзҺҮгҖӮ

1.3.1 дёҖеұӨзҡ„ијёеҮәеңЁе№ҫдҪ•дёҠжҳҜд»ҖйәјпјҹпјҲжҸӣеә§жЁҷзі»пјӢйқһз·ҡжҖ§ж‘әз–Ҡпјү

еүҚйқўи«Үзҡ„жҳҜгҖҢдёҖеұӨзҡ„з·ҡжҖ§йғЁеҲҶ $\mathbf{W}\mathbf{x}+\mathbf{b}$гҖҚгҖӮдҪҶзңҹжӯЈйӨөзөҰдёӢдёҖеұӨзҡ„пјҢжҳҜеҶҚз¶“йҒҺжҝҖжҙ» $\mathbf{a}=\sigma(\mathbf{z})$ зҡ„ијёеҮәгҖӮйӮЈеҖӢијёеҮәеҗ‘йҮҸ $\mathbf{a}$ еңЁе№ҫдҪ•дёҠд»ЈиЎЁд»Җйәјпјҹз”ұж·әеҲ°ж·ұжңүеӣӣеұӨж„Ҹзҫ©пјҡ

  1. жҜҸеҖӢеҲҶйҮҸпјқдёҖж¬ЎжҠ•еҪұпјҲе°ҚйҪҠзЁӢеәҰпјүгҖӮ еҲҶйҮҸ $z_j=\mathbf{w}_j\!\cdot\!\mathbf{x}+b_j$ зҡ„е№ҫдҪ•ж„Ҹзҫ©пјҲ1.1пјүжҳҜжҠҠијёе…Ҙй»һ $\mathbf{x}$ жҠ•еҪұеҲ°ж–№еҗ‘ $\mathbf{w}_j$пјҢйҮҸгҖҢ$\mathbf{x}$ жңүеӨҡе°ҚйҪҠз¬¬ $j$ еҖӢеӯёеҲ°зҡ„зү№еҫөжЁЈжқҝгҖҚпјӣзӯүеғ№ең°пјҢ$z_j$ жӯЈжҜ”ж–ј $\mathbf{x}$ еҲ°гҖҢз¬¬ $j$ еҖӢзҘһз¶“е…ғжұәзӯ–и¶…е№ійқўгҖҚзҡ„её¶иҷҹи·қйӣўгҖӮ
  2. ж•ҙеҖӢијёеҮәеҗ‘йҮҸпјқиіҮж–ҷй»һеңЁгҖҢж–°зү№еҫөеә§жЁҷзі»гҖҚзҡ„ж–°еә§жЁҷпјҲre-embeddingпјүгҖӮ еҺҹжң¬ $\mathbf{x}$ з”ЁеҺҹе§Ӣзү№еҫөи»ёжҸҸиҝ°пјӣдёҖеұӨд№ӢеҫҢж”№з”ЁгҖҢе°ҚжЁЈжқҝ 1 зҡ„е°ҚйҪҠеәҰгҖҒе°ҚжЁЈжқҝ 2 зҡ„е°ҚйҪҠеәҰвҖҰгҖҚйҖҷзө„ж–°и»ёдҫҶжҸҸиҝ°еҗҢдёҖеҖӢй»һзҡ„дҪҚзҪ®гҖӮжүҖд»ҘдёҖеұӨзҡ„ијёеҮәпјқйӮЈеҖӢй»һеңЁгҖҢз”ұеӯёеҲ°зҡ„ж–№еҗ‘ејөжҲҗзҡ„ж–°з©әй–“гҖҚиЈЎзҡ„ж–°еә§жЁҷпјӣдёӢдёҖеұӨзңӢеҲ°зҡ„дёҚжҳҜеҺҹе§ӢиіҮж–ҷпјҢиҖҢжҳҜйҖҷд»ҪйҮҚж–°иЎЁйҒ”йҒҺзҡ„еә§жЁҷгҖӮ
  3. йқһз·ҡжҖ§ $\sigma$пјқжҠҠз©әй–“ж‘әз–ҠпјҸжүӯжӣІпјҲйҖҷжҳҜгҖҢж·ұеәҰгҖҚжңүж•Ҳзҡ„й—ңйҚөпјүгҖӮ иӢҘжІ’жңү $\sigma$пјҢеӨҡеұӨеҸӘжҳҜзҹ©йҷЈйҖЈд№ҳпјқд»ҚжҳҜдёҖеҖӢзҹ©йҷЈпјҲ1.3пјүпјҢе№ҫдҪ•дёҠеҸӘжңүж—ӢиҪүпјӢзё®ж”ҫпјӢеүӘеҲҮпјҢеҶҚеӨҡеұӨд№ҹеҲҮдёҚй–Ӣжң¬дҫҶз·ҡжҖ§дёҚеҸҜеҲҶзҡ„иіҮж–ҷгҖӮ$\sigma$пјҲReLUгҖҒsigmoidпјүжңғжҠҠеә§жЁҷз©әй–“ж‘әиө·дҫҶгҖҒеЈ“жүҒгҖҒеҪҺжҠҳпјҢж–јжҳҜеҺҹжң¬зәҸеңЁдёҖиө·зҡ„е…©йЎһиў«жӢүеҲ°гҖҢдёҖеҲҖпјҲдёӢдёҖеұӨи¶…е№ійқўпјүе°ұиғҪеҲҶй–ӢгҖҚзҡ„дҪҚзҪ®гҖӮ
  4. з¶ӯеәҰи®ҠеҢ–пјқеҚҮз¶ӯеұ•й–ӢпјҸйҷҚз¶ӯеЈ“зё®гҖӮ зҘһз¶“е…ғж•ё > ијёе…Ҙз¶ӯеәҰ вҶ’ еҚҮз¶ӯж”Өй–ӢпјҲйЎһдјј kernel trickпјҢжң¬дҫҶж“ дёҖиө·зҡ„жӣҙеҘҪеҲҶпјүпјӣзҘһз¶“е…ғж•ё < ијёе…Ҙз¶ӯеәҰ вҶ’ йҷҚз¶ӯеЈ“зё®пјҢеҸӘз•ҷжңҖжңүеҲӨеҲҘеҠӣзҡ„зү№еҫөеә§жЁҷгҖӮ

ж‘әзҙҷжҜ”е–»гҖӮ дёҖејөз•«и‘—е…©жўқдәӨзәҸиһәж—Ӣзҡ„зҙҷпјҢж”Өе№іжҷӮдёҖжўқзӣҙз·ҡеҲҶдёҚй–ӢпјӣжҠҠзҙҷж‘әе№ҫж‘әеҫҢпјҢдёҖеҲҖе°ұиғҪеҲҮй–ӢгҖӮжҜҸдёҖеұӨзҡ„ијёеҮәпјҢе°ұжҳҜйӮЈејөзҙҷиў«ж‘әйҒҺдёҖж¬ЎеҫҢзҡ„ж–°зӢҖж…ӢгҖӮ

дёҖеҸҘи©ұзёҪзөҗпјҡеүҚдёҖеұӨзҡ„ијёеҮәпјқеҗҢдёҖеҖӢиіҮж–ҷй»һиў«йҮҚж–°иЎЁйҒ”жҲҗгҖҢеңЁдёҖзө„еӯёеҲ°зҡ„ж–№еҗ‘дёҠеҗ„жҠ•еҪұеӨҡе°‘гҖҚзҡ„ж–°еә§жЁҷпјҢиҖҢдё”ж•ҙеҖӢз©әй–“е·Іиў«йқһз·ҡжҖ§ж‘әйҒҺдёҖж¬ЎпјҢеҘҪи®“дёӢдёҖеұӨзҡ„и¶…е№ійқўжӣҙе®№жҳ“жҠҠйЎһеҲҘеҲҮй–ӢгҖӮйҖҷе°ұжҳҜгҖҢж·ұеәҰгҖҚзҡ„ж„Ҹзҫ©вҖ”вҖ”жҜҸеӨҡдёҖеұӨпјҢе°ұеӨҡж‘әдёҖж¬ЎгҖҒжҸӣдёҖж¬ЎжӣҙжңүеҲ©зҡ„еә§жЁҷзі»пјҲжҺҘеӣһ 1.2 зҘһз¶“е…ғгҖҒеүҚжҺҘдё»йЎҢ 8 еүҚеҗ‘еӮіж’ӯпјүгҖӮ

1.4 еёёз”ЁзҜ„ж•ёиҲҮзӣёдјјеәҰ

е…ҲеҫһгҖҢй•·еәҰгҖҚиҲҮгҖҢи·қйӣўгҖҚзҡ„е№ҫдҪ•зӣҙиҰәи¬ӣиө·гҖӮ дёҖеҖӢеҗ‘йҮҸеҸҜзңӢжҲҗеҫһеҺҹй»һжҢҮеҗ‘жҹҗй»һзҡ„з®ӯй ӯгҖӮжңҖиҮӘз„¶зҡ„е…©еҖӢйҮҸжҳҜпјҡй•·еәҰпјқйҖҷж”Ҝз®ӯй ӯжңүеӨҡй•·пјҲеҺҹй»һеҲ°йӮЈеҖӢй»һеӨҡйҒ пјүпјӣи·қйӣўпјқе…©еҖӢй»һзӣёйҡ”еӨҡйҒ пјҢд№ҹе°ұжҳҜе·®еҗ‘йҮҸ $\mathbf{a}-\mathbf{b}$ зҡ„й•·еәҰгҖӮж—Ҙеёёз®—й•·еәҰз”Ёзҡ„жҳҜзӣҙз·ҡи·қйӣўпјҲз•ўж°Ҹе®ҡзҗҶпјү$\sqrt{x_1^2+x_2^2+\dots}$ вҖ” йіҘйЈӣзҡ„жңҖзҹӯзӣҙз·ҡпјҢе№ҫдҪ•дёҠжҳҜдёҖжўқзӣҙз·ҡж®өгҖӮ

дҪҶгҖҢзӣҙз·ҡи·қйӣўгҖҚдёҚжҳҜе”ҜдёҖеҗҲзҗҶзҡ„йҮҸжі•гҖӮ еңЁжҹҗдәӣжғ…жіҒпјҢз”Ёзӣҙз·ҡи·қйӣўдҫҶжҜ”ијғдёҰдёҚжҒ°з•¶жҲ–дёҚж–№дҫҝпјҢйңҖиҰҒжҸӣдёҖзЁ®гҖҢжҖҺйәјз®—й•·еәҰгҖҚзҡ„иҰҸеүҮпјҡ

  • еёӮеҚҖжЈӢзӣӨж јиЎ—йҒ“дёҚиғҪиө°е°Қи§’з·ҡгҖҒеҸӘиғҪжІҝиЎ—иҪүзӣҙи§’ вҶ’ еҜҰйҡӣи·қйӣўжҳҜиө°йҒҺзҡ„ж јеӯҗзёҪй•·пјҢдёҚжҳҜзӣҙз·ҡгҖӮ
  • жңүжҷӮеҸӘеңЁд№Һи®ҠеҢ–жңҖеӨ§зҡ„йӮЈдёҖз¶ӯпјҲдҫӢеҰӮгҖҢжҜҸеҖӢиҰҸж јйғҪдёҚиғҪи¶…жЁҷгҖҚпјүвҶ’ и©Із”ЁгҖҢеҸ–жңҖеӨ§йӮЈдёҖз¶ӯгҖҚз•¶и·қйӣўгҖӮ
  • иіҮж–ҷжңүйӣўзҫӨеҖјпјҢжҲ–жғіеҸӘзңӢж–№еҗ‘дёҚзңӢй•·зҹӯпјҲжҜ”дё»йЎҢеғҸдёҚеғҸпјүвҶ’ зӣҙз·ҡи·қйӣўйғҪдёҚжҳҜжңҖеҘҪзҡ„йҒёж“ҮгҖӮ

д№ҹе°ұжҳҜиӘӘпјҢгҖҢй•·еәҰпјҸи·қйӣўгҖҚе…¶еҜҰжҳҜдёҖзЁ®еҸҜд»ҘйҒёж“Үзҡ„е®ҡзҫ© вҖ” дҫқдҪ жғіжҜ”ијғд»ҖйәјгҖҒеңЁд№Һд»ҖйәјиҖҢдёҚеҗҢгҖӮзӮәдәҶиғҪеңЁеҫҢйқўе…¬е№іең°жҜ”ијғгҖҒжҢ‘еҲ°йҒ©еҗҲзҡ„еәҰйҮҸпјҢж•ёеӯёе®¶жҠҠгҖҢдёҖзЁ®еҗҲзҗҶзҡ„й•·еәҰйҮҸжі•гҖҚжҠҪиұЎеҮәдҫҶпјҢеҸ–еҗҚеҸ«зҜ„ж•ё (norm)пјӣеҗҢдёҖзө„еҗ‘йҮҸжҸӣдёҖеҖӢзҜ„ж•ёпјҢе°ұжҸӣдёҖзЁ®гҖҢжҖҺйәјз®—й•·еәҰпјҸи·қйӣўгҖҚзҡ„иҰҸеүҮгҖӮ

зҜ„ж•ёзҡ„е®ҡзҫ©гҖӮ дёҖзЁ®гҖҢй•·еәҰйҮҸжі•гҖҚ $\lVert\cdot\rVert$пјҲжҠҠеҗ‘йҮҸжҳ еҲ°йқһиІ еҜҰж•ёпјүиҰҒеӨ ж јеҸ«зҜ„ж•ёпјҢеҝ…й Ҳж»ҝи¶ідёүжўқе…¬зҗҶпјҡ

  • йқһиІ жҖ§пјҡ$\lVert\mathbf{x}\rVert\ge 0$пјҢдё” $=0\iff\mathbf{x}=\mathbf{0}$пјҲеҸӘжңүйӣ¶еҗ‘йҮҸй•·еәҰзӮә 0пјүгҖӮ
  • йҪҠж¬ЎжҖ§пјҡ$\lVert\alpha\mathbf{x}\rVert=|\alpha|\,\lVert\mathbf{x}\rVert$пјҲж”ҫеӨ§е№ҫеҖҚгҖҒй•·еәҰе°ұе№ҫеҖҚпјүгҖӮ

жңҖеёёиҰӢзҡ„е…ҘеҸЈжҳҜ PCAпјҡз¶Іи·ҜйҒҷжё¬еёёжңүж•ёеҚҒеҖӢеҪјжӯӨзӣёй—ңзҡ„зү№еҫөпјҢPCA з”Ёзү№еҫөеҲҶи§ЈжүҫеҮәгҖҢзңҹжӯЈжңүи®ҠеҢ–зҡ„е°‘ж•ёдё»ж–№еҗ‘гҖҚпјҢжҠҠз¶ӯеәҰйҷҚдёӢдҫҶгҖҒеҺ»жҺүеҶ—йӨҳ вҖ” йҖҷе°ұжҳҜзү№еҫөеҖј / еҗ‘йҮҸеңЁ ML зҡ„ж—Ҙеёёз”ЁйҖ”пјҢд№ҹе’Ңдё»йЎҢ 8 Embedding зҡ„гҖҢз”Ёе°‘ж•ёз¶ӯеәҰиЎЁеҫөиіҮж–ҷгҖҚдёҖи„ҲзӣёжүҝгҖӮ

е…ҲжҗһжҮӮгҖҢе…ұи®Ҡз•°ж•ёзҹ©йҷЈ (covariance matrix)гҖҚжҳҜд»ҖйәјгҖӮ е®ғз”ұе…©еҖӢжӣҙеҹәжң¬зҡ„йҮҸзө„жҲҗпјҲз¬Ұиҷҹпјҡ$x_i,y_i$пјқз¬¬ $i$ зӯҶиіҮж–ҷеңЁе…©еҖӢзү№еҫөпјҲfeatureпјүдёҠзҡ„еҖјпјҢйқһ eigenvalueпјӣ$\bar x,\bar y$пјқе…©зү№еҫөзҡ„е№іеқҮпјҢ$\bar x=\frac1n\sum_i x_i$пјӣ$n$пјқиіҮж–ҷй»һж•ёпјҸжЁЈжң¬ж•ёпјүпјҡ

  • и®Ҡз•°ж•ё $\text{Var}(x)=\frac{1}{n-1}\sum_i(x_i-\bar x)^2$пјҡе–®дёҖзү№еҫөгҖҢж•ЈеӨҡй–ӢгҖҚгҖӮ
  • е…ұи®Ҡз•°ж•ё $\text{Cov}(x,y)=\frac{1}{n-1}\sum_i(x_i-\bar x)(y_i-\bar y)$пјҡе…©зү№еҫөгҖҢжҳҜеҗҰдёҖиө·еӢ•гҖҚгҖӮй—ңйҚөзңӢд№ҳз©Қзҡ„жӯЈиІ  вҖ” жҹҗй»һзҡ„ $x,y$ еҗҢеңЁеҗ„иҮӘе№іеқҮзҡ„еҗҢдёҖеҒҙпјҲйғҪеҒҸеӨ§жҲ–йғҪеҒҸе°ҸпјүвҶ’ жӯЈпјӣдёҚеҗҢеҒҙ вҶ’ иІ гҖӮе№іеқҮеҫҢпјҡCov $\gt$ 0 жӯЈзӣёй—ңпјҲдёҖиө·жјІпјүгҖҒCov $\lt$ 0 иІ зӣёй—ңпјҲжӯӨж¶ҲеҪјй•·пјүгҖҒ$\approx$ 0 з„Ўй—ңгҖӮ

жҠҠе®ғеҖ‘жҺ’жҲҗиЎЁе°ұжҳҜе…ұи®Ҡз•°ж•ёзҹ©йҷЈпјҲ$d$ еҖӢзү№еҫө вҶ’ $d\times d$пјүпјҡе°Қи§’з·ҡ = еҗ„зү№еҫөзҡ„и®Ҡз•°ж•ёгҖҒйқһе°Қи§’з·ҡ = е…©е…©е…ұи®Ҡз•°ж•ёпјӣеӣ  $\text{Cov}(x,y)=\text{Cov}(y,x)$ иҖҢе°ҚзЁұпјҡ

$$\mathbf{C}=\begin{bmatrix}\text{Var}(x) & \text{Cov}(x,y)\\ \text{Cov}(x,y) & \text{Var}(y)\end{bmatrix}$$

пјҲйҷӨд»Ҙ $n-1$ жҳҜжЁЈжң¬и®Ҡз•°ж•ёзҡ„жЁҷжә–еҒҡжі• вҖ” иІқзҙўж ЎжӯЈпјҢеӣ е№іеқҮ $\bar x$ з”ұеҗҢжү№иіҮж–ҷдј°еҮәгҖҒжңғи®“йӣўе·®е№іж–№е’Ңз•ҘеҒҸе°ҸпјӣиӢҘжҠҠиіҮж–ҷз•¶жҲҗж•ҙеҖӢжҜҚй«”еүҮйҷӨд»Ҙ $n$гҖӮпјүе®ғжҠҠиіҮж–ҷйӣІзҡ„еҪўзӢҖиҲҮеӮҫж–ңз·ЁзўјжҲҗдёҖеҖӢзҹ©йҷЈпјҲе°Қи§’з·ҡ = еҗ„ж–№еҗ‘ж•ЈеӨҡй–ӢгҖҒйқһе°Қи§’з·ҡ = еҫҖе“ӘеҖӢж–ңж–№еҗ‘зӣёй—ңпјүпјӣе°Қе®ғеҒҡзү№еҫөеҲҶи§Је°ұеҫ—еҲ°йӣІзҡ„дё»и»ё вҖ” йҖҷе°ұжҳҜдёӢйқўзҡ„ PCAгҖӮ

  2. е…ұи®Ҡз•°ж•ёзҹ©йҷЈпјҡеҲҶеҲҘз®— $\text{Var}(x)$гҖҒ$\text{Var}(y)$гҖҒ$\text{Cov}(x,y)$ еҶҚеЎ«йҖІзҹ©йҷЈпјҲз”ЁдёӯеҝғеҢ–еҫҢзҡ„ $x,y$пјҢйҷӨд»Ҙ $n-1=4$пјүпјҡ$\text{Var}(x)=\frac{2^2+1^2+0+(-1)^2+(-2)^2}{4}=\frac{10}{4}=2.5$пјӣ$\text{Var}(y)=\frac{1^2+2^2+0+(-2)^2+(-1)^2}{4}=2.5$пјӣ$\text{Cov}(x,y)=\frac{(2)(1)+(1)(2)+0+(-1)(-2)+(-2)(-1)}{4}=\frac{8}{4}=2$пјҲвҶҗ йҖҷиЈЎжүҚеҗҢжҷӮз”ЁеҲ° $x$ иҲҮ $y$пјүпјҡ $$\mathbf{C}=\begin{bmatrix}\text{Var}(x)&\text{Cov}(x,y)\\ \text{Cov}(x,y)&\text{Var}(y)\end{bmatrix}=\begin{bmatrix}2.5 & 2\\ 2 & 2.5\end{bmatrix}$$ пјҲз·Ҡж№ҠеҜ«жі•пјҡ$\mathbf{C}=\frac{1}{n-1}\sum_i(\mathbf{x}_i-\bar{\mathbf{x}})(\mathbf{x}_i-\bar{\mathbf{x}})^\top$пјҢе…¶дёӯ $\mathbf{x}_i=(x_i,y_i)^\top$ жҳҜж•ҙзӯҶиіҮж–ҷзҡ„еҗ‘йҮҸгҖҒе…©йӮҠжҳҜеҗҢдёҖеҖӢеҗ‘йҮҸпјӣеӨ–з©ҚжҷӮд№ҳжі•жңғжҠҠ $x$ еҲҶйҮҸй…Қ $y$ еҲҶйҮҸпјҢиҮӘеӢ•еңЁйқһе°Қи§’з·ҡз”ҹеҮә $x{\cdot}y$ е…ұи®Ҡз•°ж•ёй … вҖ” жүҖд»ҘгҖҢе…©йӮҠйғҪжҳҜ $\mathbf{x}$гҖҚжҳҜе°Қзҡ„пјҢ$y$ и—ҸеңЁеҗ‘йҮҸиЈЎгҖӮпјү
  3. зү№еҫөеҲҶи§Јпјҡи§Јзү№еҫөж–№зЁӢеҫ— $\lambda_1=4.5,\ \lambda_2=0.5$пјҢе°ҚжҮүзү№еҫөеҗ‘йҮҸ $\mathbf{v}_1=\frac{1}{\sqrt2}(1,1)$гҖҒ$\mathbf{v}_2=\frac{1}{\sqrt2}(1,-1)$пјҲиЁҲз®—зҙ°йғЁиҰӢдёӢпјүгҖӮ

(b) зү№еҫөеҗ‘йҮҸ вҖ” жҜҸеҖӢ $\lambda$ д»Јеӣһ $(\mathbf{C}-\lambda\mathbf{I})\mathbf{v}=\mathbf{0}$ и§ЈгҖӮ

$$\mathbf{C}\begin{bmatrix}1\\1\end{bmatrix}=\begin{bmatrix}2.5+2\\2+2.5\end{bmatrix}=\begin{bmatrix}4.5\\4.5\end{bmatrix}=4.5\begin{bmatrix}1\\1\end{bmatrix},\qquad \mathbf{C}\begin{bmatrix}1\\-1\end{bmatrix}=\begin{bmatrix}0.5\\-0.5\end{bmatrix}=0.5\begin{bmatrix}1\\-1\end{bmatrix}$$

еӣ зӮә $\mathbf{C}$ е°ҚзЁұпјҢзү№еҫөеҗ‘йҮҸеҝ…дә’зӣёеһӮзӣҙ вҶ’ дё»и»ёжҳҜдёҖзө„жӯЈдәӨж–°еә§жЁҷзі»пјҲе‘јжҮү 1.3.1 жҸӣеә§жЁҷзі»пјүгҖӮе№ҫдҪ•дёҠ $\mathbf{C}$ жҠҠиіҮж–ҷйӣІз·ЁзўјжҲҗдёҖеҖӢжІҝ $(1,1)$ жӢүй•·зҡ„ж©ўең“пјҡй•·и»ёж–№еҗ‘ = $\mathbf{v}_1$гҖҒеҚҠи»ёй•·ВІ = зү№еҫөеҖјгҖӮ

е·ҘзЁӢж„Ҹж¶өгҖӮ гҖҢжЁЎеһӢжҺЁи«–гҖҚеңЁж•ёеӯёдёҠ вүҲ дёҖдёІзҹ©йҷЈд№ҳжі•гҖӮжЁЎеһӢеӨ§е°ҸпјҲеҸғж•ёйҮҸпјүвүҲ жүҖжңү $\mathbf{W}$ зҡ„е…ғзҙ ж•ёпјӣжҺЁи«–з®—еҠӣ вүҲ зҹ©йҷЈд№ҳжі•ж¬Ўж•ё (FLOPs)гҖӮйҖҷзӣҙжҺҘе°ҚжҮүгҖҢжЁЎеһӢи¶ҠеӨ§и¶Ҡеҗғ VRAM иҲҮз®—еҠӣгҖҚгҖӮ

2. ж©ҹзҺҮиҲҮзөұиЁҲ вҖ” еҲҶеёғгҖҒжўқд»¶ж©ҹзҺҮгҖҒжңҖеӨ§дјјз„¶

еҲӨеҲҘејҸжЁЎеһӢијёеҮәгҖҢж©ҹзҺҮгҖҚпјҢз”ҹжҲҗејҸжЁЎеһӢеӯёгҖҢеҲҶеёғгҖҚпјҢйғҪе»әз«ӢеңЁйҖҷиЈЎгҖӮ

2.1 ж©ҹзҺҮеҲҶеёғиҲҮжңҹжңӣ

йҡЁж©ҹи®Ҡж•ё $X$ зҡ„еҲҶеёғжҸҸиҝ°еҗ„зөҗжһңзҡ„еҸҜиғҪжҖ§гҖӮжңҹжңӣеҖј (е№іеқҮ)пјҡ

$$\mathbb{E}[X]=\sum_x x\,P(x)\ (\text{йӣўж•Ј})\qquad \mathbb{E}[X]=\int x\,p(x)\,dx\ (\text{йҖЈзәҢ})$$

и®Ҡз•°ж•ё $\mathrm{Var}(X)=\mathbb{E}[(X-\mathbb{E}[X])^2]$ иЎЎйҮҸйӣўж•ЈзЁӢеәҰгҖӮ

е№ҫдҪ•ж„Ҹзҫ©гҖӮ дёҖеҖӢеҲҶеёғе°ұжҳҜдёҖжўқгҖҢж©ҹзҺҮеҜҶеәҰжӣІз·ҡгҖҚпјҢ$\mu$ жҳҜе®ғзҡ„дёӯеҝғгҖҒ$\sigma$ жҳҜиғ–зҳҰгҖӮжңҖеӨ§дјјз„¶ (MLE) е°ұжҳҜз§»еӢ•дёҰзё®ж”ҫйҖҷжўқжӣІз·ҡпјҢи®“и§Җжё¬еҲ°зҡ„иіҮж–ҷй»һзӣЎйҮҸиҗҪеңЁй«ҳзҡ„ең°ж–№гҖӮ

2.2 жўқд»¶ж©ҹзҺҮиҲҮиІқж°Ҹ

$$P(y\mid x)=\frac{P(x\mid y)\,P(y)}{P(x)}$$

  • еҲӨеҲҘејҸжЁЎеһӢзӣҙжҺҘеӯё $P(y\mid x)$пјҲзөҰе®ҡзү№еҫөпјҢйЎһеҲҘзҡ„ж©ҹзҺҮпјүвҖ” еҲҶйЎһеҷЁе°ұеңЁдј°йҖҷеҖӢгҖӮ
  • з”ҹжҲҗејҸжЁЎеһӢеӯё $P(x)$ жҲ– $P(x\mid\text{prompt})$пјҲиіҮж–ҷеҰӮдҪ•з”ўз”ҹпјүвҖ” жүҖд»ҘиғҪгҖҢз”ҹжҲҗгҖҚгҖӮ

2.3 жңҖеӨ§дјјз„¶дј°иЁҲ (MLE)

гҖҢиЁ“з·ҙгҖҚеңЁзөұиЁҲдёҠеӨҡеҚҠжҳҜжңҖеӨ§дјјз„¶пјҡжүҫ $\theta$ и®“и§Җжё¬иіҮж–ҷеҮәзҸҫж©ҹзҺҮжңҖеӨ§гҖӮ

$$\theta^\star=\arg\max_\theta \prod_{i=1}^{N} P(y_i\mid x_i;\theta)$$

йҖЈд№ҳдёҚеҘҪз®—пјҢеҸ–е°Қж•ёи®ҠйҖЈеҠ гҖҒеҠ иІ иҷҹи®ҠжңҖе°ҸеҢ–пјҡ

$$\theta^\star=\arg\min_\theta\; -\sum_{i=1}^{N}\log P(y_i\mid x_i;\theta)$$

йҖҷеҖӢ $-\sum\log P$ е°ұжҳҜиІ е°Қж•ёдјјз„¶пјҢд№ҹе°ұжҳҜеҲҶйЎһжңҖеёёз”Ёзҡ„дәӨеҸүзҶөжҗҚеӨұпјҲдё»йЎҢ 5пјүгҖӮжҗҚеӨұеҮҪж•ёеӨҡеҚҠжҳҜиІ е°Қж•ёдјјз„¶зҡ„еҢ–иә«пјҢдёҚжҳҜжҶ‘з©әиЁӯиЁҲгҖӮ

е·ҘзЁӢж„Ҹж¶өгҖӮ еҲҶйЎһеҷЁијёеҮәзҡ„жҳҜж©ҹзҺҮ/дҝЎеҝғеҲҶж•ёиҖҢйқһзЎ¬жЁҷзұӨпјӣиіҮе®үеҸҜз”Ёе®ғиӘҝгҖҢйқҲж•ҸеәҰ vs иӘӨе ұгҖҚй–ҖжӘ»пјҲдё»йЎҢ 19 ROCпјүгҖӮжҮӮ MLE е°ұжҮӮзӮәд»ҖйәјдәӨеҸүзҶөжҳҜй җиЁӯжҗҚеӨұгҖӮ

3. еҫ®з©ҚеҲҶиҲҮжўҜеәҰ вҖ” е°Һж•ёгҖҒеҒҸе°ҺгҖҒйҸҲйҺ–еҫӢ

иЁ“з·ҙ = жңҖе°ҸеҢ–жҗҚеӨұ $L(\theta)$гҖӮжҖҺйәјеҫҖжӣҙе°Ҹиө°пјҹйқ жўҜеәҰгҖӮ

3.1 е°Һж•ёиҲҮеҒҸе°Һ

е°Һж•ё $\frac{dL}{d\theta}$ жҳҜи®ҠеҢ–зҺҮпјҲж–ңзҺҮпјүгҖӮеӨҡеҸғж•ёжҷӮе°ҚжҜҸеҖӢеҸғж•ёеҸ–еҒҸе°Һ $\frac{\partial L}{\partial\theta_j}$пјҢз–ҠжҲҗеҗ‘йҮҸеҚіжўҜеәҰпјҡ

$$\nabla_\theta L=\left(\frac{\partial L}{\partial\theta_1},\frac{\partial L}{\partial\theta_2},\dots\right)^\top$$

жўҜеәҰжҢҮеҗ‘гҖҢ$L$ дёҠеҚҮжңҖеҝ«гҖҚзҡ„ж–№еҗ‘пјҢж•…дёӢйҷҚеҫҖеҸҚж–№еҗ‘иө° вҖ” йҖҷжҳҜжўҜеәҰдёӢйҷҚзҡ„е…ЁйғЁзӣҙиҰәпјҡ

$$\theta\leftarrow\theta-\eta\,\nabla_\theta L$$

е№ҫдҪ•ж„Ҹзҫ©гҖӮ жҠҠжҗҚеӨұжғіжҲҗдёҖеҖӢгҖҢзў—гҖҚпјҲеұұи°·пјүпјҢзҸҫеңЁз«ҷеңЁеұұеқЎдёҠжҹҗй»һгҖӮжўҜеәҰжҢҮеҗ‘дёҠеқЎжңҖйҷЎж–№еҗ‘пјҢеҸ–иІ иҷҹе°ұжҳҜжңҖйҷЎдёӢеқЎпјӣжҜҸжӯҘиө° $\eta$пјҲеӯёзҝ’зҺҮпјүеӨ§е°ҸпјҢйҖҗжӯҘж»ҫеҗ‘и°·еә•пјҲжңҖе°ҸжҗҚеӨұпјүгҖӮ$\eta$ еӨӘеӨ§жңғиЎқйҒҺй ӯгҖҒеӨӘе°ҸжңғеҫҲж…ўгҖӮ

3.2 йҸҲйҺ–еҫӢ вҖ” еҸҚеҗ‘еӮіж’ӯзҡ„еј•ж“Һ

зҘһз¶“з¶Іи·ҜжҳҜеұӨеұӨиӨҮеҗҲеҮҪж•ё $L=f_k(f_{k-1}(\cdots f_1(\mathbf{x})))$гҖӮиҰҒз®— $L$ е°ҚеүҚеұӨж¬ҠйҮҚзҡ„еҒҸе°ҺпјҢз”ЁйҸҲйҺ–еҫӢйҖҗеұӨзӣёд№ҳпјҡ

$$\frac{dL}{dw}=\frac{dL}{da}\cdot\frac{da}{dz}\cdot\frac{dz}{dw}$$

дҫӢеҰӮ $L=(y-\hat y)^2$гҖҒ$\hat y=\sigma(z)$гҖҒ$z=wx+b$пјҢеүҮ

$$\frac{\partial L}{\partial w}=\underbrace{2(\hat y-y)}_{\partial L/\partial\hat y}\cdot\underbrace{\sigma'(z)}_{\partial\hat y/\partial z}\cdot\underbrace{x}_{\partial z/\partial w}$$

еҸҚеҗ‘еӮіж’ӯе°ұжҳҜжҠҠйҸҲйҺ–еҫӢз”ұијёеҮәеҫҖијёе…ҘйҖҗеұӨеҘ—з”ЁпјҲдё»йЎҢ 10 е®Ңж•ҙжҺЁе°ҺпјүгҖӮеұҖйғЁе°Һж•ёеңЁеүҚеҗ‘жҷӮиЁҳдёӢгҖҒеҸҚеҗ‘жҷӮзӣёд№ҳпјҢи®“зҷҫиҗ¬еҸғж•ёзҡ„жўҜеәҰдёҖж¬Ўз®—еҮәгҖӮ

е·ҘзЁӢж„Ҹж¶өгҖӮ гҖҢиЁ“з·ҙеҗғз®—еҠӣйҒ еӨ§ж–јжҺЁи«–гҖҚзҡ„ж•ёеӯёеҺҹеӣ пјҡжҺЁи«–еҸӘеҒҡеүҚеҗ‘пјҲдёҖйҒҚзҹ©йҷЈд№ҳжі•пјүпјӣиЁ“з·ҙиҰҒеүҚеҗ‘ + еҸҚеҗ‘пјҲеҶҚз®—жўҜеәҰпјү+ жӣҙж–°пјҢдё”йҮҚиӨҮеҫҲеӨҡијӘгҖӮе°ҚжҮүгҖҢйӮҠз·ЈеҒҡжҺЁи«–гҖҒеҶҚиЁ“з·ҙж”ҫдёӯеӨ®гҖҚгҖӮ

4. з·ҡжҖ§еӣһжӯёиҲҮжҗҚеӨұеҮҪж•ё

жңҖе°Ҹзҡ„гҖҢжңғеӯёзҝ’гҖҚжЁЎеһӢпјҢжҠҠеүҚдёүз« дёІиө·дҫҶгҖӮ

4.1 жЁЎеһӢиҲҮжҗҚеӨұ

з·ҡжҖ§еӣһжӯёеҒҮиЁӯ $\hat y=\mathbf{w}^\top\mathbf{x}+b$гҖӮз”ЁеқҮж–№иӘӨе·® (MSE) з•¶жҗҚеӨұпјҡ

$$L(\mathbf{w},b)=\frac{1}{N}\sum_{i=1}^{N}\big(\hat y_i-y_i\big)^2=\frac{1}{N}\sum_{i=1}^{N}\big(\mathbf{w}^\top\mathbf{x}_i+b-y_i\big)^2$$

MSE е…¶еҜҰжҳҜгҖҢиӘӨе·®жңҚеҫһй«ҳж–ҜеҲҶеёғгҖҚдёӢзҡ„жңҖеӨ§дјјз„¶пјҲдё»йЎҢ 2пјүвҖ” еҶҚж¬ЎеҚ°иӯүжҗҚеӨұдҫҶиҮӘж©ҹзҺҮеҒҮиЁӯгҖӮ

е№ҫдҪ•ж„Ҹзҫ©гҖӮ еңЁж•Јй»һең–дёҠжүҫдёҖжўқзӣҙз·ҡпјҢи®“жҜҸеҖӢй»һеҲ°з·ҡзҡ„гҖҢеһӮзӣҙи·қйӣўгҖҚе№іж–№е’ҢжңҖе°ҸгҖӮеһӮзӣҙи·қйӣўе°ұжҳҜж®ҳе·® $\hat y_i - y_i$пјӣе№іж–№жҳҜзӮәдәҶжҮІзҪ°еӨ§иӘӨе·®гҖҒдё”е°ҚжҮүй«ҳж–ҜеҒҮиЁӯгҖӮ

еҲҶйЎһ vs еӣһжӯёпјҲдёҖејөең–зңӢжҮӮпјүгҖӮ еҗҢдёҖе ҶиіҮж–ҷй»һпјҡеҲҶйЎһжҳҜжӢҝи¶…е№ійқўжҠҠй»һйӣІгҖҢеҲҮй–ӢгҖҚпјҲеҲӨеңЁе“ӘдёҖеҒҙ/е“ӘдёҖеҚҖпјүпјӣеӣһжӯёжҳҜжӢҝдёҖжўқз·ҡгҖҢз©ҝйҒҺгҖҚй»һйӣІпјҲи®“й»һзӣЎйҮҸиІјиҝ‘гҖҒй җжё¬ж•ёеҖјпјүгҖӮе…©иҖ…йғҪе»әз«ӢеңЁ $\mathbf{w}^\top\mathbf{x}+b$ дёҠпјҢе·®еҲҘеҸӘеңЁијёеҮәжҖҺйәјз”Ё вҖ” еҲҶйЎһзңӢиҗҪеңЁе“ӘдёҖеҒҙпјҲargmaxпјүпјҢеӣһжӯёзӣҙжҺҘи®Җж•ёеҖјгҖӮ

4.2 жўҜеәҰиҲҮжӣҙж–°

$$\nabla_{\mathbf{w}}L=\frac{2}{N}\sum_{i=1}^{N}\big(\hat y_i-y_i\big)\,\mathbf{x}_i$$

еҶҚз”ЁжўҜеәҰдёӢйҷҚ $\mathbf{w}\leftarrow\mathbf{w}-\eta\nabla_{\mathbf{w}}L$гҖӮгҖҢй җжё¬иӘӨе·® $(\hat y-y)$ д№ҳијёе…Ҙ $\mathbf{x}$гҖҚзҡ„еҪўејҸжңғеңЁзҘһз¶“з¶Іи·ҜеҸҚиҰҶеҮәзҸҫ вҖ” backprop жңҖеӨ–еұӨж°ёйҒ жҳҜиӘӨе·®й …гҖӮ

е·ҘзЁӢж„Ҹж¶өгҖӮ з¶Іи·Ҝ AI зҡ„гҖҢй »еҜ¬/еҒҘеә·й җжё¬гҖҚжң¬иіӘе°ұжҳҜеӣһжӯёпјӣз·ҡжҖ§еӣһжӯёжҳҜзҗҶи§ЈдёҖеҲҮзҡ„иө·й»һгҖӮMSE йҒ©еҗҲйҖЈзәҢеҖјпјҢдёҚйҒ©еҗҲеҲҶйЎһпјҲдёӢзҜҖжҸӣжҗҚеӨұпјүгҖӮ

5. йӮҸијҜеӣһжӯёгҖҒSigmoidгҖҒSoftmax иҲҮдәӨеҸүзҶө

еҲҶйЎһжЁЎеһӢзҡ„ж•ёеӯёж ёеҝғ вҖ” е№ҫд№ҺжүҖжңүеҲҶйЎһеҷЁпјҲеҗ«зҘһз¶“з¶Іи·ҜијёеҮәеұӨпјүйғҪз”ЁйҖҷеҘ—гҖӮ

5.1 дәҢе…ғеҲҶйЎһпјҡSigmoid + дәҢе…ғдәӨеҸүзҶө

иҰҒијёеҮәж©ҹзҺҮ (0~1)пјҢжҠҠз·ҡжҖ§ијёеҮә $z=\mathbf{w}^\top\mathbf{x}+b$ йҒҺ sigmoidпјҡ

$$\sigma(z)=\frac{1}{1+e^{-z}}\in(0,1)$$

е№ҫдҪ•ж„Ҹзҫ©гҖӮ Sigmoid жҳҜдёҖжўқ S еҪўжӣІз·ҡпјҢжҠҠж•ҙжўқеҜҰж•ёз·ҡгҖҢеЈ“гҖҚйҖІ (0,1)пјҡ$z=0$ жҷӮијёеҮә 0.5пјҲжұәзӯ–иҮЁз•ҢпјүпјҢ$z$ еҫҲеӨ§и¶Ёиҝ‘ 1гҖҒеҫҲе°Ҹи¶Ёиҝ‘ 0гҖӮеүӣеҘҪеҸҜз•¶гҖҢеұ¬ж–јжӯЈйЎһзҡ„ж©ҹзҺҮгҖҚгҖӮ

$\hat y=\sigma(z)$ и§Ји®ҖзӮә $P(y=1\mid x)$гҖӮжҗҚеӨұз”ЁдәҢе…ғдәӨеҸүзҶө (BCE)пјҲеҚіиІ е°Қж•ёдјјз„¶пјүпјҡ

$$L=-\frac{1}{N}\sum_{i=1}^{N}\big[y_i\log\hat y_i+(1-y_i)\log(1-\hat y_i)\big]$$

жјӮдә®иҷ•пјҡ$\sigma'(z)=\sigma(z)(1-\sigma(z))$пјҢдё” BCE е°Қ $z$ зҡ„жўҜеәҰеҢ–з°ЎжҲҗ $\frac{\partial L}{\partial z}=\hat y-y$ вҖ” еҸҲжҳҜгҖҢй җжё¬жёӣзңҹеҖјгҖҚпјҢд№ҫж·ЁгҖҒж•ёеҖјз©©е®ҡгҖӮ

5.2 еӨҡе…ғеҲҶйЎһпјҡSoftmax + дәӨеҸүзҶө

$C$ йЎһжҷӮпјҢијёеҮәеұӨзөҰ $C$ еҖӢеҲҶж•ё (logits) $\mathbf{z}=(z_1,\dots,z_C)$пјҢз”Ё softmax иҪүжҲҗж©ҹзҺҮеҲҶеёғпјҡ

$$\text{softmax}(\mathbf{z})_k=\frac{e^{z_k}}{\sum_{j=1}^{C} e^{z_j}}$$

еҗ„й … 0~1 дё”зёҪе’ҢзӮә 1гҖӮжҗҚеӨұз”ЁйЎһеҲҘдәӨеҸүзҶөпјҲ$y$ зӮә one-hotгҖҒзңҹйЎһзӮә $t$пјүпјҡ

$$L=-\sum_{k=1}^{C} y_k\log\hat y_k=-\log\hat y_t$$

еҚігҖҢзңҹйЎһж©ҹзҺҮзҡ„иІ е°Қж•ёгҖҚвҖ” и¶ҠзўәдҝЎжӯЈзўәйЎһгҖҒжҗҚеӨұи¶Ҡе°ҸгҖӮжўҜеәҰеҗҢжЁЈеҢ–з°ЎзӮә $\frac{\partial L}{\partial z_k}=\hat y_k-y_k$гҖӮ

е№ҫдҪ•ж„Ҹзҫ©пјҲеӨҡйЎһжҖҺйәјгҖҢеҲҮгҖҚпјүгҖӮ дәҢе…ғеҲҶйЎһжҳҜгҖҢдёҖеҖӢи¶…е№ійқўеҲҮжҲҗе…©йӮҠгҖҚпјӣеӨҡйЎһдёҚжҳҜйҖЈзәҢдәҢеҲҶпјҢиҖҢжҳҜ $C$ еҖӢеҲҶж•ёеҗ„иҮӘжҜ”й«ҳдҪҺгҖҒеҸ–жңҖеӨ§пјҲargmaxпјүпјҡ

  • жҜҸеҖӢ logit $z_k=\mathbf{w}_k\cdot\mathbf{x}+b_k$ жҳҜ $\mathbf{x}$ е°ҚгҖҢз¬¬ $k$ йЎһжЁЈжқҝ $\mathbf{w}_k$гҖҚзҡ„е°ҚйҪҠзЁӢеәҰпјҲжҺҘеӣһ 1.1пјүпјӣеҲҶйЎһпјқжҢ‘жңҖе°ҚйҪҠзҡ„йЎһпјҡ$\hat y=\arg\max_k z_k$гҖӮ
  • е№ҫдҪ•дёҠйҖҷжҠҠз©әй–“еҲҮжҲҗ $C$ еҖӢеҚҖеҹҹпјҡжҜҸй»һжӯёзөҰеҲҶж•ёжңҖй«ҳзҡ„йЎһпјӣзӣёй„°е…©йЎһзҡ„йӮҠз•ҢжҳҜгҖҢеҲҶж•ёзӣёзӯү $z_i=z_j$гҖҚзҡ„и¶…е№ійқўгҖӮз·ҡжҖ§ softmax жҷӮеҚҖеҹҹзӮәеҮёеӨҡйӮҠеҪўгҖҒйӮҠз•ҢзӮәзӣҙз·ҡж®өгҖӮ
  • еүҚйқўе№ҫеұӨиӢҘе·Із”Ё $\sigma$ ж‘әз–ҠйҒҺз©әй–“пјҲ1.3.1пјүпјҢйҖҷдәӣйӮҠз•ҢеңЁеҺҹе§Ӣз©әй–“е°ұиғҪд»»ж„ҸеҪҺжӣІпјҢеҲҮеҮәд»»дҪ•еҪўзӢҖгҖҒд»»ж„ҸеӨҡйЎһгҖӮ

5.3 ж•ёеҖјз©©е®ҡпјҡsoftmax жёӣ max

$e^{z_k}$ еңЁ $z$ еӨ§жҷӮжәўдҪҚгҖӮеҜҰдҪңе…ҲжёӣжңҖеӨ§еҖјпјҲзөҗжһңдёҚи®Ҡпјүпјҡ

$$\text{softmax}(\mathbf{z})_k=\frac{e^{z_k-\max_j z_j}}{\sum_j e^{z_j-\max_j z_j}}$$

е·ҘзЁӢж„Ҹж¶өгҖӮ Softmax жҳҜжүҖжңүеҲҶйЎһеҷЁиҲҮ LLM ијёеҮәеұӨзҡ„е…ұеҗҢзөӮй»һ вҖ” з¶Іи·ҜжөҒйҮҸеҲҶйЎһијёеҮәгҖҢеҗ„йЎһеҲҘж©ҹзҺҮгҖҚгҖҒLLM ијёеҮәгҖҢдёӢдёҖеҖӢ token зҡ„ж©ҹзҺҮеҲҶеёғгҖҚз”Ёзҡ„йғҪжҳҜе®ғгҖӮжҮӮ softmax + дәӨеҸүзҶөпјҢзӯүж–јжҺҢжҸЎеҲӨеҲҘејҸеҲҶйЎһиҲҮз”ҹжҲҗејҸеҸ–жЁЈпјҲдё»йЎҢ 17пјүзҡ„е…ұеҗҢеҹәеә§гҖӮ

7. жӯЈеүҮеҢ– (Regularization) вҖ” е°ҚжҠ—йҒҺж“¬еҗҲ

дё»йЎҢ 6пјҲжўҜеәҰдёӢйҷҚпјҸе„ӘеҢ–еҷЁпјүеҫ…иЈңпјӣжң¬зҜҖе…Ҳе»әз«ӢжӯЈеүҮеҢ–иҰҒе°Қд»ҳзҡ„е•ҸйЎҢ вҖ” йҒҺж“¬еҗҲгҖӮ

7.1 йҒҺж“¬еҗҲ vs ж¬ ж“¬еҗҲ

йҒҺж“¬еҗҲ (overfitting)пјҡжЁЎеһӢжҠҠиЁ“з·ҙиіҮж–ҷгҖҢиғҢйҒҺй ӯгҖҚпјҢйҖЈиіҮж–ҷиЈЎзҡ„йӣңиЁҠгҖҒе·§еҗҲйғҪеӯёйҖІеҺ» вҶ’ иЁ“з·ҙйӣҶеҲҶж•ёи¶…й«ҳгҖҒдҪҶйҒҮеҲ°жІ’зңӢйҒҺзҡ„ж–°иіҮж–ҷе°ұе·®гҖӮе®ғеӯёеҲ°зҡ„жҳҜгҖҢйҖҷжү№иіҮж–ҷзҡ„зү№дҫӢгҖҚпјҢдёҚжҳҜгҖҢйҖҡеүҮгҖҚгҖӮе°ұеғҸеӯёз”ҹжӯ»иғҢиҖғеҸӨйЎҢпјҢжҸӣеҖӢйЎҢзӣ®е°ұдёҚжңғгҖӮ

ж¬ ж“¬еҗҲ (underfitting)пјҡеҸҚйҒҺдҫҶпјҢжЁЎеһӢеӨӘз°Ўе–®пјҢйҖЈиЁ“з·ҙиіҮж–ҷзҡ„и¶ЁеӢўйғҪжҠ“дёҚеҲ° вҶ’ иЁ“з·ҙгҖҒжё¬и©ҰйғҪе·®гҖӮ

жЁЎеһӢиӨҮйӣңеәҰе№ҫдҪ•иЁ“з·ҙжё¬и©Ұ
ж¬ ж“¬еҗҲеӨӘз°Ўе–®зӣҙз·ҡзЎ¬еҘ—еҪҺжӣІиіҮж–ҷгҖҒжҠ“дёҚеҲ°и¶ЁеӢўе·®е·®
еүӣеҘҪйҒ©дёӯе№іж»‘жӣІз·ҡиІјеҗҲи¶ЁеӢўгҖҒе®№иЁұжҜҸй»һжңүй»һиӘӨе·®еҘҪеҘҪ
йҒҺж“¬еҗҲеӨӘиӨҮйӣңжүӯжӣІжӣІз·ҡзЎ¬з©ҝжҜҸеҖӢй»һгҖҒйҖЈйӣңиЁҠйғҪеӯёжҘөеҘҪе·®

жҖҺйәјеҲӨж–·пјҡиЁ“з·ҙиӘӨе·®дҪҺгҖҒдҪҶй©—иӯүпјҸжё¬и©ҰиӘӨе·®й«ҳпјҢе…©иҖ…иҗҪе·® (gap) еӨ§ = йҒҺж“¬еҗҲзҡ„дҝЎиҷҹгҖӮеҺҹеӣ йҖҡеёёжҳҜжЁЎеһӢеӨӘиӨҮйӣңпјҲеҸғж•ёеӨӘеӨҡпјүгҖҒиіҮж–ҷеӨӘе°‘гҖҒжҲ–иЁ“з·ҙеӨӘд№…гҖӮ

жҖҺйәјйҳІпјҲпјқжӯЈеүҮеҢ–зҡ„д»»еӢҷпјүпјҡи®“жЁЎеһӢгҖҢеҲҘжҠҠиЁҠиҷҹиЁҳеӨӘжӯ»гҖҚвҖ”вҖ”

  • L1 / L2 жӯЈеүҮеҢ–пјҡжҮІзҪ°ж¬ҠйҮҚеӨ§е°Ҹ $\lVert\mathbf{w}\rVert$пјҲз”Ё 1.4 зҡ„зҜ„ж•ёзҗғйҷҗеҲ¶ж¬ҠйҮҚпјүпјҢйҖјжЁЎеһӢз”Ёжӣҙе°ҸпјҸжӣҙе°‘зҡ„ж¬ҠйҮҚгҖӮ
  • е…¶д»–пјҡжӣҙеӨҡиіҮж–ҷгҖҒearly stoppingпјҲж—©й»һеҒңпјүгҖҒdropoutгҖҒз°ЎеҢ–жЁЎеһӢгҖҒдәӨеҸүй©—иӯүгҖӮ

е·ҘзЁӢж„Ҹж¶өгҖӮ Demo жҷӮгҖҢиЁ“з·ҙжә–зўәзҺҮ 99%гҖҚеҸҜиғҪеҸӘжҳҜиғҢдәҶйӮЈжү№жЁЈжң¬пјҢжҸӣзңҹеҜҰжөҒйҮҸе°ұеӨұж•ҲвҖ”вҖ”йҖҷе°ұжҳҜйҒҺж“¬еҗҲгҖӮй©—ж”¶йӮҠз·ЈжЁЎеһӢиҰҒз”ЁжҢүжҷӮй–“еҲҮгҖҒеҗ«зңҹеҜҰжҜ”дҫӢжғЎж„ҸжЁЈжң¬зҡ„жё¬и©ҰйӣҶпјҢзңӢе®ғе°ҚгҖҢжІ’зңӢйҒҺзҡ„иіҮж–ҷгҖҚйӮ„жә–дёҚжә–гҖӮ

йҷ„йҢ„ A вҖ” йӨҳејҰе®ҡеҫӢ (Law of Cosines)

е…§з©Қзҡ„д»Јж•ёе®ҡзҫ©иҲҮе№ҫдҪ•е®ҡзҫ©зӮәдҪ•зӣёзӯү (дё»йЎҢ 1.1)пјҹй—ңйҚөжҳҜйӨҳејҰе®ҡеҫӢгҖӮжң¬йҷ„йҢ„е…ҲзөҰйӨҳејҰе®ҡеҫӢзҡ„ж•ҳиҝ° (A.1) иҲҮиҮӘи¶іжҺЁе°Һ (A.2пјҢдёҚдҪҝз”Ёе…§з©Қ)пјҢеҶҚз”Ёе®ғиӯүжҳҺе…§з©Қе…©зЁ®е®ҡзҫ©зӯүеғ№ (A.3)гҖӮ

A.1 ж•ҳиҝ°

дёүи§’еҪўдёүйӮҠй•· $a, b, c$пјҢиЁӯ $C$ зӮәйӮҠ $c$ жүҖе°Қзҡ„еӨҫи§’пјҢеүҮ

$$c^2 = a^2 + b^2 - 2ab\cos C$$

  • жҳҜз•ўж°Ҹе®ҡзҗҶзҡ„жҺЁе»Јпјҡз•¶ $C=90^\circ$пјҢ$\cos C=0$ вҶ’ $c^2=a^2+b^2$пјҲз•ўж°Ҹе®ҡзҗҶпјүгҖӮйӨҳејҰе®ҡеҫӢеӨҡдәҶдҝ®жӯЈй … $-2ab\cos C$гҖӮ
  • $C \lt 90^\circ$пјҲйҠіи§’пјүвҶ’ $\cos C \gt 0$ вҶ’ $c^2 \lt a^2+b^2$пјӣ$C \gt 90^\circ$пјҲйҲҚи§’пјүвҶ’ $\cos C \lt 0$ вҶ’ $c^2 \gt a^2+b^2$гҖӮеӨҫи§’и¶ҠеӨ§пјҢе°ҚйӮҠи¶Ҡй•·гҖӮ

A.2 жҺЁе°ҺпјҲеә§жЁҷжі•пјҢжңҖд№ҫж·Ёпјү

жҠҠеӨҫи§’ $C$ ж”ҫеңЁеҺҹй»һгҖҒйӮҠ $b$ жІҝ x и»ёпјҡ$C=(0,0)$гҖҒ$A=(b,0)$гҖҒ$B=(a\cos C,\ a\sin C)$гҖӮ

йӮҠ $c=\lvert AB\rvert$пјҢз”Ёи·қйӣўе…¬ејҸпјҡ

$$c^2 = (b - a\cos C)^2 + (0 - a\sin C)^2$$

еұ•й–Ӣпјҡ

$$c^2 = b^2 - 2ab\cos C + a^2\cos^2 C + a^2\sin^2 C$$

еӣ  $\cos^2 C + \sin^2 C = 1$пјҲз•ўж°ҸжҒҶзӯүејҸпјүпјҢ$a^2\cos^2 C + a^2\sin^2 C = a^2$пјҡ

$$\boxed{\;c^2 = a^2 + b^2 - 2ab\cos C\;}\qquad\blacksquare$$

жӯӨжҺЁе°ҺеҸӘз”ЁдәҶи·қйӣўе…¬ејҸ + з•ўж°ҸжҒҶзӯүејҸпјҢе®Ңе…ЁзҚЁз«Ӣж–је…§з©ҚпјҢжүҖд»ҘдёӢйқў A.3гҖҢз”ЁйӨҳејҰе®ҡеҫӢиӯүе…§з©ҚгҖҚдёҚж§ӢжҲҗеҫӘз’°и«–иӯүгҖӮпјҲзӯүеғ№дҪңжі•пјҡеҫһ $B$ е°ҚйӮҠ $b$ дҪңеһӮз·ҡгҖҒз”Ёз•ўж°Ҹе®ҡзҗҶеҲҶж®өзӣёеҠ пјҢзөҗжһңзӣёеҗҢгҖӮпјү

A.3 з”ЁйӨҳејҰе®ҡеҫӢиӯүжҳҺе…§з©Қзҡ„е…©зЁ®е®ҡзҫ©зӯүеғ№

е…§з©Қжңүе…©еҖӢе®ҡзҫ© вҖ” д»Јж•ёзҡ„гҖҢеҲҶйҮҸд№ҳз©Қд№Ӣе’Ң $\sum_i a_ib_i$гҖҚиҲҮе№ҫдҪ•зҡ„гҖҢ$\lVert\mathbf{a}\rVert\lVert\mathbf{b}\rVert\cos\phi$гҖҚгҖӮз”Ё A.2 зҡ„йӨҳејҰе®ҡеҫӢеҸҜиӯүе®ғеҖ‘зӣёзӯүгҖӮеҸ–з”ұ $\mathbf{a}$гҖҒ$\mathbf{b}$ иҲҮе…¶е·® $\mathbf{a}-\mathbf{b}$ еңҚжҲҗзҡ„дёүи§’еҪўпјҲеӨҫи§’ $\phi$пјүпјҡ

(1) д»Јж•ёеұ•й–ӢпјҲзҙ”з”ЁеҲҶйҮҸгҖҒз•ўж°Ҹпјүпјҡ

$$\lVert\mathbf{a}-\mathbf{b}\rVert^2=\sum_i (a_i-b_i)^2=\sum_i a_i^2-2\sum_i a_ib_i+\sum_i b_i^2=\lVert\mathbf{a}\rVert^2+\lVert\mathbf{b}\rVert^2-2\sum_i a_ib_i$$

(2) йӨҳејҰе®ҡеҫӢпјҲA.2пјҢдёүйӮҠ $\lVert\mathbf{a}\rVert$гҖҒ$\lVert\mathbf{b}\rVert$гҖҒ$\lVert\mathbf{a}-\mathbf{b}\rVert$пјҢеӨҫи§’ $\phi$пјүпјҡ

$$\lVert\mathbf{a}-\mathbf{b}\rVert^2=\lVert\mathbf{a}\rVert^2+\lVert\mathbf{b}\rVert^2-2\lVert\mathbf{a}\rVert\lVert\mathbf{b}\rVert\cos\phi$$

е…©ејҸе·ҰйӮҠеҗҢзӮә $\lVert\mathbf{a}-\mathbf{b}\rVert^2$пјҢеҸійӮҠзӣёзӯүпјӣж¶ҲеҺ» $\lVert\mathbf{a}\rVert^2+\lVert\mathbf{b}\rVert^2$ еҫ—пјҡ

$$\boxed{\;\sum_i a_ib_i=\lVert\mathbf{a}\rVert\lVert\mathbf{b}\rVert\cos\phi\;}$$

еҚігҖҢеҲҶйҮҸд№ҳз©Қд№Ӣе’ҢгҖҚжҒ°зӯүж–јгҖҢе…©еҗ‘йҮҸй•·еәҰ Г— еӨҫи§’йӨҳејҰгҖҚ вҖ” д»Јж•ёиҲҮе№ҫдҪ•е…©зЁ®е…§з©Қе®ҡзҫ©е…¶еҜҰжҳҜеҗҢдёҖд»¶дәӢпјҢйҖҷд№ҹи§ЈйҮӢдәҶзӮәд»ҖйәјгҖҢеҲҶйҮҸзЎ¬д№ҳзӣёеҠ гҖҚжңғзӯүж–јгҖҢйҮҸжё¬ж–№еҗ‘е°ҚйҪҠзЁӢеәҰгҖҚгҖӮ
